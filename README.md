## 🌟 О проекте

QuickTab — это мощный инструмент для мгновенного получения важной информации через терминал, голосовые команды и современную графическую оболочку. Создан для тех, кто хочет получать данные без лишних кликов и переключения вкладок.

### 🎯 Зачем нужен QuickTab?

Каждый день мы тратим время на:
- 🔍 Проверку погоды в браузере
- 💰 Поиск курсов валют и криптовалют
- 📰 Просмотр новостей и актуальных событий
- ❓ Быстрые ответы на вопросы
- 🔄 Переключение между множеством вкладок

**QuickTab решает это одной командой, голосом или кликом!**

## ✨ Реализованные функции

### 🖥️ Графическая оболочка (CustomTkinter)
- 🎨 Современный GUI с темной темой и цветными акцентами
- 📊 Дашборд с виджетами: погода, валюты, новости
- 🔍 Умный поиск с краткой сводкой и веб-ссылками
- 🎙️ Голосовое управление (в разработке)
- ⚙️ Настраиваемый размер окон и интерфейс
- 📱 Адаптивный дизайн для разных разрешений

### 🌤️ Виджеты и данные
- **Погода**: Температура, ветер, влажность, прогноз
- **Валюты**: Курсы USD, EUR, BTC с конвертером
- **Новости**: Кибербезопасность, политика, экономика, технологии
- **Поиск**: Краткие ответы + веб-поиск (DuckDuckGo, Wikipedia)

### 🔍 Умный поиск с ИИ

**Google Gemini AI доступен прямо в приложении:**
- 🤖 **ИИ-ответы**: "Объясни теорию относительности", "Как приготовить борщ?", "Расскажи о древних цивилизациях"
- 🔄 **Режим уточнения**: Нажмите "Задать уточнение" для углубленного диалога по теме
- 💡 **Краткие сводки**: "Погода в Москве" → мгновенный ответ
- 🧮 **Калькулятор**: "2+2*3" → результат
- 🌐 **Веб-поиск**: Ссылки на DuckDuckGo
- 📚 **Wikipedia**: Ответы на вопросы знаний
- 💱 **Конвертер**: "100 долларов в рубли"

## 🛠️ Быстрый старт

### 📥 Установка

```bash
# 1. Скачай проект
git clone https://github.com/yaroslav-belov101/QuickTab.git
cd QuickTab

# 2. Создай виртуальное окружение (рекомендуется)
python -m venv env
source env/bin/activate  # Linux/Mac
# или env\Scripts\activate  # Windows

# 3. Установи зависимости
pip install -r requirements.txt
```

### 🌐 Браузерные драйверы

#### Windows (автоматически):
```cmd
pip install webdriver-manager
```
**Всё! Драйверы скачаются автоматически.**

#### Linux:
```bash
# Ubuntu/Debian
sudo apt install firefox-geckodriver chromium-browser

# Arch/Manjaro
sudo pacman -S geckodriver chromium

# Fedora
sudo dnf install geckodriver chromium
```

#### macOS:
```bash
brew install geckodriver chromedriver
```

### ▶️ Запуск

```bash
cd QuickTab
python main.py
```

**Увидишь красивый GUI с виджетами погоды, валют и новостей!**


## 🎉 Что происходит?

1. **🦊 Firefox** открывает твой обычный профиль с куками
2. Загружает Яндекс.Погода Белореченск 
3. **Парсит** температуру/ветер/влажность
4. **Обновляет** каждые 60 секунд
5. **Ctrl+C** → мгновенно закрывается

## Почему QuickTab?

- **Firefox default** = ваши cookies, расширения, настройки
- **Regex парсинг** = работает при любых изменениях сайта
- **Fallback Chromium** = 100% стабильность
- **Модульность** = легко добавлять новые виджеты
- **Три интерфейса** = терминал + голос + GUI
